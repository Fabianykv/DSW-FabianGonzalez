"Propósito del sitio"

El propósito de este sitio es aprender mediante problemas de la vida real a comprender de una mejor manera temas como las etiquetas semánticas, CSS Grid

Etiquetas Semánticas

En los nuevos estándares de la utilización del leguaje HTML5, para la creación de páginas más accesibles para todo el público, por esto ha incluido las etiquetas semánticas con el fin de hacer más sencillo el código a la hora de ser presentado, en decir que estás servirán para dividir un documento en partes lógicas, indicando el tipo de contenido que se incluye en cada parte, con estas se mejora la accesibilidad para toda clase de persona que se enfrenta con el código.

Diez, T., Domínguez, M. J., Martínez, J. J., & Sáenz, J. (2012). Creación de páginas Web accesibles con HTML5. Consultado el, 26. Recuperado de //www.esvial.org/wp-content/files/Atica2012_pp120-129.pdf

CSS Grid

Como es conocido el CSS en un lenguaje que se integra con HTML5, en este caso el CSS Grid es un sistema de diseño web que no ayuda a dividir la pagina en una cuadricula, por lo cual nos ayuda a distribuir los elementos en una forma más sencilla.

Atkins Jr, T., Etemad, EJ y Atanassov, R. (2013). Diseño de cuadrícula CSS. Recuperado de https://www.w3.org/TR/2013/WD-css3-grid-layout-20130402/
